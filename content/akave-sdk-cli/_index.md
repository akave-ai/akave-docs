The **Akave CLI (`akavecli`)** is the official command-line interface for interacting with Akave's decentralized storage network. It provides developers with tools for bucket and file management, efficient data operations, and seamless integration with Akave's blockchain-based architecture. To get started, review the [Akave CLI Getting Started]({{< relref "akave-sdk-cli/akavecli-getting-started.md" >}}) section.
